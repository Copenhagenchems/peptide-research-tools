## Peptide Reconstitution Calculator (JS)

Simple calculator for reconstitution math:

```js
// Usage: node peptide-calculator.js
const result = calculate({ vialMg: 5, waterMl: 2, doseMcg: 250 });
console.log(result);
// → { concentration: '2500 mcg/ml', unitsPerDose: 10, dosesPerVial: 20 }
```

Open `peptide-calculator.js` in a browser or run with Node.js.
